# 🎯 링코 출석체크

링코 동호회를 위한 스마트 출석 관리 시스템

## ✨ 주요 기능

### 👥 회원 관리
- 회원 추가/삭제 (관리자 전용)
- 150명 이상 대규모 회원 관리 가능
- 이름순 자동 정렬

### ⚡ 빠른 출석 체크
- 이름만 입력하면 즉시 출석 처리
- 실시간 자동완성 기능
- 출석 현황 한눈에 확인

### 🔍 검색 및 필터
- 회원 이름 실시간 검색
- 전체/출석/미출석 필터링
- 대규모 회원도 쉽게 관리

### 💾 자동 저장
- 출석 체크 시 자동으로 저장
- 별도의 저장 버튼 불필요
- 날짜별 출석 기록 관리

### 📊 통계 및 분석
- 실시간 출석률 계산
- 회원별 출석 통계
- 날짜별 출석 현황

### 📱 모바일 최적화
- 모든 화면 크기 대응
- 터치 친화적 인터페이스
- 반응형 디자인

### 🔐 관리자 모드
- 비밀번호: **0227**
- 회원 추가/삭제 권한
- 엑셀 내보내기 권한

### 📥 엑셀 내보내기 (관리자 전용)
3개의 시트로 구성:
1. **날짜별 출석 현황** - 참가자별 행 분리, 필터 기능
2. **회원별 통계** - 출석률, 최근 출석일
3. **회원별 출석 상세** - 전체 출석 이력

## 🚀 사용 방법

### 관리자
1. 우측 상단 "🔒 관리자 모드" 클릭
2. 비밀번호 입력: `0227`
3. 회원 추가
4. 정기적으로 엑셀 백업

### 일반 사용자
1. 빠른 출석 체크에 이름 입력
2. 또는 회원 목록에서 직접 클릭
3. 자동으로 저장됨

## 💾 데이터 관리

### 데이터 저장 위치
- 각 사용자의 브라우저 (LocalStorage)
- 서버에 저장되지 않음
- 개인 정보 보호

### 백업 권장
- 정기적으로 엑셀로 내보내기
- 중요한 데이터는 별도 보관
- 브라우저 데이터 삭제 시 주의

## ⚠️ 주의사항

### 데이터가 유지되는 경우

### 데이터가 사라지는 경우

❌ 다른 브라우저/기기 사용
❌ 시크릿 모드 사용
